##### Image

```html
import MaterialDesignSymbol

var symbol:MaterialDesignSymbol = MaterialDesignSymbol(text:MaterialDesignIcon.list48px, size:25)
var iconImage:UIImage = symbol.imageWithSize(CGSizeMake(25, 25))
```
